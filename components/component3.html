Cards

The card component includes an image at the top, followed by a project header and a small description underneath. Additionally, there is a button labeled "View Project" located below the description.

Accessibility

The card component is designed with clear visual hierarchy and alt text for images to ensure accessibility for users with visual impairments.

Details

The card component is utilized to showcase individual projects on the website. It provides a visually appealing layout for presenting project details, including images, titles, and descriptions. The "View Project" button offers users a clear call-to-action to explore more about each project.
